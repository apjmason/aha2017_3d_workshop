Session 1

The first session will provide an introduction to 3D modeling and Virtual Reality, and an overview of key terminology and basic concepts.

Our demo project for the course will be a virtual museum. We chose this project since it includes many elements common to any 3D or virtual reality project, but in a cultural heritage application that should be of interest to humanists.

Once you have installed Unity, download the project files by clicking on the zipped folders at the top of the page. Extract the files into the Unity directory and double-click on the demo-scene to launch.

Session 2

In this session we will highlight some resources for finding downloadable 3D models, and introduce the basics of 3D modeling with SketchUp.

  • Sketchfab.com is one of the best communities for uploading and sharing 3D models. Some are for game design and animation, but many are accurate scans of artifacts made using photogrammetry from cultural heritage institutions like the British Museum
  • The Smithosonian X3D project contains a fantastic viewer for high quality scans, some of which are downloadable.
  • The Virtual Hampson Museum was an early and exemplary model for a digitized public, downloadable museum collection.
  • 3D Warehouses contain models others have created, some available to download for free, many paid
    • SketchUp 3D Warehouse
    • The Unity Asset Store
    • Turbosquid
